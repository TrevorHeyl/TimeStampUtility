# TimeStampUtility

A GUI utility to convert to <-> from Date/Time to Unix or Y2K epoch, Developed in C# Under Visual Studio 2013

Resulution in seconds.

The Epochs are 2000/01/01 00:00:00(Y2K)  and 1970/01/01 00:00:00
